Welcome to our site!

This is a simple blog website built to demonstrate some features of the Flask server framework for Python.

So far, we've added the following features:

  • A database using the flask_sqlalchemy library.
  • A page listing every stored blog article, with a link to view the article's full text in a separate page.
  • A password-protected admin page, which allows the admin user to add, edit and delete blog posts
